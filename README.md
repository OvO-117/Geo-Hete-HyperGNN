## Geo-Hete-HyperGNN
Geo-Hete-HyperGNN: Novo Molecular Hypergraph Representation Learning strategy

## 依赖环境 (Dependencies)

本项目基于 Python 开发，主要依赖 PyTorch 深度学习框架及化学信息学相关库。

### 🛠️ 环境要求

- **Python**:3.10.19
- **CUDA**: 11.8 
- **PyTorch**:2.1.1+cu118
- **networkx**: 3.4.2
- **rdkit**: 2025.3.6
- **torch_geometric**:2.5.1
